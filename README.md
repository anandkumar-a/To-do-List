# To-Do List Application

## 📌 Project Overview

The To-Do List Application is a simple task management application developed to help users organize and track their daily activities efficiently. Users can create, update, complete, and delete tasks through an easy-to-use interface.

This project was built to strengthen concepts in Java development, CRUD operations, and application design.

## 🚀 Features

* Add new tasks
* View all tasks
* Update existing tasks
* Mark tasks as completed
* Delete tasks
* Manage daily activities efficiently

## 🛠 Technologies Used

* Java
* Spring Framework / Spring Boot
* MySQL
* Hibernate / JPA
* Maven
* REST API
* Postman
* IntelliJ IDEA / Eclipse

## 📋 Task Details Managed

The application stores information such as:

* Task ID
* Task Title
* Task Description
* Task Status (Pending/Completed)
* Created Date
* Updated Date

## ⚙️ API Endpoints

| Method | Endpoint               | Description              |
| ------ | ---------------------- | ------------------------ |
| GET    | `/tasks`               | Retrieve all tasks       |
| GET    | `/tasks/{id}`          | Retrieve a task by ID    |
| POST   | `/tasks`               | Create a new task        |
| PUT    | `/tasks/{id}`          | Update an existing task  |
| DELETE | `/tasks/{id}`          | Delete a task            |
| PATCH  | `/tasks/{id}/complete` | Mark a task as completed |

## ▶️ How to Run the Project

1. Clone the repository.
2. Configure the database credentials in `application.properties`.
3. Start the MySQL server.
4. Build the project using Maven.
5. Run the application.
6. Access the APIs using Postman or your frontend application.
